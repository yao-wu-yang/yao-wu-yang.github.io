This is my report of Lab3

我使用了github Page+hexo的组合来完成个人博客的搭建:
1.配置Node.js
2.安装git
3.通过ssh与github进行链接 最终成功部署
准备工作：确保你已经拥有了一个 GitHub 账号，并在本地安装了 Git 和 Node.js。
安装 Hexo：使用 npm 在本地安装 Hexo，Hexo 是一个基于 Node.js 的静态博客框架，它可以帮助你快速搭建和部署博客网站。
创建 Hexo 站点：使用 Hexo 命令创建一个新的博客站点，并选择一个喜欢的主题。
配置 Hexo 站点：根据个人需求，编辑 Hexo 站点的配置文件，包括主题配置、博客信息、社交链接等。
编写博文：使用 Markdown 格式编写博文，并保存在 Hexo 站点的 source/_posts 目录下。
生成静态网页：使用 Hexo 命令生成静态网页，将 Markdown 文件转换为 HTML 页面。
部署到 GitHub Pages：将生成的静态网页部署到 GitHub Pages，你可以选择使用 hexo-deployer-git 插件来简化部署过程。
自定义域名（可选）：如果你有自己的域名，可以通过修改 GitHub Pages 设置来将博客网站绑定到自定义域名上。
